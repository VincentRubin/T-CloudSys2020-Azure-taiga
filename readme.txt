You will need the following informations to run the scripts :

Azure tenant ID
Azure client ID
Azure client SECRET
Azure Subscription ID

export AZURE_TENANT_ID={your tenant id}
export AZURE_CLIENT_ID={your client id}
export AZURE_CLIENT_SECRET={your client secret}
export AZURE_SUBSCRIPTION_ID={your subscription id}


More informations here : 
https://docs.microsoft.com/en-us/samples/azure-samples/resource-manager-python-template-deployment/resource-manager-python-template-deployment/


To run : python azure_deployment.py
